# Modelagem de banco de dados
Este repositório reúne os trabalhos desenvolvidos ao longo do semestre na disciplina **Modelagem de Banco de Dados**.

## Objetivo dos Projetos

## ✅ 1. Entender como os dados são organizados

Você aprende a criar e modelar estruturas como tabelas, colunas, relacionamentos, chaves primárias/estrangeiras, etc.

## ✅ 2. Aprender linguagens de consulta (SQL)

A disciplina ensina a usar SQL para:

-inserir dados

-buscar informações

-atualizar registros

-excluir dados

-criar e alterar estruturas no banco

## ✅ 3. Garantir integridade e segurança dos dados

##Conceitos como:

-normalização

-constraints

-controle de concorrência

-backup e recuperação

## ✅ 4. Melhorar desempenho

Você aprende sobre índices, otimização de consultas e como deixar o banco rápido mesmo com muitos dados.

## ✅ 5. Entender diferentes modelos de banco

### Como:

 Relacionais (MySQL, PostgreSQL, SQL Server)
 
-Não relacionais (MongoDB, Redis, Cassandra)

-Data warehouses e bancos para análise

## ✅ 6. Aplicar em sistemas reais

### A disciplina prepara para criar bancos usados em:

-sistemas de vendas

-aplicativos web

-aplicativos móveis

-jogos

-sistemas financeiros

plataformas de streaming

e praticamente qualquer software que use dados
