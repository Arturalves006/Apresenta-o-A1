# Algoritmos e pensamento computacional

Este repositório reúne os trabalhos desenvolvidos ao longo do semestre na disciplina **Algoritmos e pensamento computacional**.

## Objetivo dos Projetos

## ✅ 1. Ensinar a pensar como um programador

Você aprende a quebrar problemas grandes em partes menores, identificar padrões, criar estratégias e encontrar soluções passo a passo.

## ✅ 2. Introduzir o conceito de algoritmos

Um algoritmo é um conjunto de passos para resolver um problema.

### Na disciplina, você aprende a:

-criar algoritmos

-representar algoritmos (fluxogramas, pseudocódigo)

-analisar se um algoritmo é eficiente

## ✅ 3. Preparar para aprender programação

### Antes de aprender linguagens como Python, Java ou C, é importante entender:

-lógica

-variáveis

-condições (if)

-repetições (loops)

-funções

-Esses conceitos começam aqui.

## ✅ 4. Desenvolver habilidades de raciocínio lógico

### O pensamento computacional ajuda a resolver problemas não só na informática, mas em qualquer área, porque melhora:

-organização

-análise

-criação de estratégias

-tomada de decisão

## ✅ 5. Base para outras disciplinas

### Serve como base para:

-Programação

-Estruturas de Dados

-Banco de Dados

Inteligência Artificial

Engenharia de Software
