# Desenvolvimento FRONT-END

Este repositório reúne os trabalhos desenvolvidos ao longo do semestre na disciplina **Desenvolvimento FRONT-END**.

## Objetivo dos Projetos

## ✅ 1. Criar interfaces de usuário (UI)

Aprender a desenvolver telas, botões, menus, formulários, layouts e todo o visual de uma aplicação.

## ✅ 2. Construir páginas web usando tecnologias básicas

### As três principais tecnologias ensinadas são:

=HTML – estrutura da página

=CSS – aparência e estilos

=JavaScript – interação e comportamento

## ✅ 3. Tornar sites responsivos

 Aprender a fazer interfaces que funcionam bem em:

-celulares

-tablets

-computadores

## ✅ 4. Criar projetos reais

### Saber criar:

-sites completos

-landing pages

dashboards

interfaces de aplicativos web
